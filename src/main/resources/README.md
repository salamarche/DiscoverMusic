# Resources 

## Required resources
* ### awsCredentials.properties
  Properties used for AWS Cognito integration:
  * clientId
  * clientSecret
  * httpsJwks
  * expectedIssuer
  * loginLink
* ### hibernate.cfg.xml
  Hibernate configuration
* ### log4j2.properties
  Log4J configuration properties
* ### spotifyAPI.properties
  Properties used for spotifyAPI
    * clientId
    * clientSecret
